# 思源闪卡标题工具箱 - 项目概述

## 1. 项目简介

**思源闪卡标题工具箱** (sy-flashcard-toolbox-plugin) 是一个思源笔记插件，主要功能是管理思源笔记中的闪卡标题。该插件允许用户在编辑器视图中为闪卡设置自定义标题，并在复习界面中显示这些自定义标题。

## 2. 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue | 3.3.8 | 前端 UI 框架 |
| TypeScript | 5.0.4 | 主要开发语言 |
| Vite | 6.2.1 | 构建工具 |
| siyuan | 1.1.0 | 思源插件 SDK |
| Sass | 1.62.1 | CSS 预处理器 |

## 4. 核心功能

### 4.1 编辑器视图功能
- 在编辑器中扫描所有闪卡元素
- 为每个闪卡添加"编辑标题"按钮
- 提供对话框界面编辑闪卡的自定义标题

### 4.2 复习界面功能
- 监听复习界面的闪卡显示
- 自动将闪卡原标题替换为自定义标题
- 支持标签切换时的标题更新

## 6. 运行环境检测

插件在加载时会检测以下运行环境：

| 属性 | 说明 |
|------|------|
| `isMobile` | 是否在移动端运行 |
| `isBrowser` | 是否在浏览器中运行 |
| `isLocal` | 是否在本地运行 |
| `isElectron` | 是否在 Electron 中运行 |
| `isInWindow` | 是否在独立窗口中运行 |
