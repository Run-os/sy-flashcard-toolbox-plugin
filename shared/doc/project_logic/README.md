## 核心流程概览

```
插件加载 → 环境检测 → Vue应用初始化 → 功能模块初始化
    ↓
编辑器视图: 扫描闪卡 → 添加编辑按钮 → 保存自定义标题
    ↓
复习界面: 检测闪卡 → 读取自定义标题 → 替换显示标题
```
